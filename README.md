# Heart-disease-prediction
AI based Heart Disease Prediction System
###  Heart Disease Prediction System

A Machine Learning project developed using **Python** to predict whether a person is at risk of heart disease based on medical data.

#### ✨ Features

🔍 Predicts Heart Disease Risk
📊 Data Visualization with Charts
🤖 Machine Learning-based Prediction
🎯 Simple and User-Friendly

#### 🛠️ Tech Stack

🐍 Python
🐼 Pandas & NumPy
📈 Matplotlib & Seaborn
🤖 Scikit-learn
📓 Jupyter Notebook

#### 🎯 Outcome

Helps in the early detection of heart disease by analyzing patient health data and providing accurate predictions. ❤️🏥
